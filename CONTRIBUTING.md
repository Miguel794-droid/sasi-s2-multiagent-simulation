Thank you for your interest in contributing to SASI! We welcome contributions from researchers, developers, and safety advocates worldwide.

## How to Contribute

1. **Fork the repository**
2. **Create your feature branch**: `git checkout -b feature/AmazingFeature`
3. **Commit your changes**: `git commit -m 'Add some AmazingFeature'`
4. **Push to the branch**: `git push origin feature/AmazingFeature`
5. **Open a Pull Request**

## What We're Looking For

- **Bug fixes**: Especially in the viability calculation logic
- **New agent types**: Additional agent archetypes for complex scenarios
- **Performance improvements**: Optimizing API usage and simulation speed
- **Documentation**: Better explanations of the structural alignment mechanism
- **Testing**: Comprehensive test suites for edge cases

## Code of Conduct

All contributors must adhere to respectful collaboration. SASI is built on principles of equity, freedom, and technical honesty.
